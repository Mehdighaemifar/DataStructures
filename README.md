# DataStructures in Java

This repository contains several fundamental **data structure implementations in Java** including stacks, queues, heaps, tries, and more. It was created to help students and developers learn how common data structures are implemented from scratch and how to use them in your own projects.

---

## Contents

The following Java classes are included:

| Data Structure | File |
|----------------|------|
| AVL Tree       | `AVL.java` |
| Max Heap       | `MaxHeap.java` |
| Queue          | `Queue.java` |
| Stack          | `Stack.java` |
| Trie           | `Trie.java` |

Each class implements a specific data structure with basic operations such as insertion, deletion, and search.
